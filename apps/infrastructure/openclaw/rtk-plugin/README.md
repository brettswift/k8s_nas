# RTK plugin (vendored)

This directory vendors the OpenClaw RTK rewrite plugin from [rtk-ai/rtk](https://github.com/rtk-ai/rtk)
(`openclaw/` at tag **v0.33.1**).

The gateway image copies these files into `~/.openclaw/extensions/rtk-rewrite/` on the PVC at pod start.

Upstream docs: [openclaw/README.md](https://github.com/rtk-ai/rtk/blob/master/openclaw/README.md).
